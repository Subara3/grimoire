# Grimoire（グリモワ）

定型文＝呪文を綴じた魔導書。
クリックひとつで定型文をクリップボードに「詠唱」する、軽量なローカル管理ツールです。
MMOのショートカット／ホットバーのような感覚で定型文を呼び出せます。

**コミッションに返答するクリエイター向け。** 受注・お断り・進捗連絡・お礼・規約など、
くり返し使う文面をフォルダごとに整理し、ホットキーで呼び出してワンクリックで貼り付け。
住所や本名・振込先などの機密は鍵付き＋全データ手元保存で、外部に出さず安全に扱えます。

## 特長

- フォルダ(タブ)で分類 — 用途ごとに呪文（定型文）を整理
- クリックで詠唱 — 行をクリックするだけでクリップボードへコピー
- キーボード操作 — ↑↓で選択・Enterでコピー・1〜9で即コピー・Escで隠す
- 直接貼り付け（任意） — コピー後、呼び出す前のアプリへ自動で Ctrl+V（設定でオン/オフ。既定オフ）
- 本文が主役のカード表示 — 貼り付ける文章を大きく表示。ドラッグで並び替え
- お気に入り(★) — よく使う定型文を★で登録。各フォルダの上部に固定され、「お気に入り」タブにも集約
- 全フォルダ横断検索 — 検索すると現在のフォルダだけでなく全フォルダから絞り込み
- ホバーで全文プレビュー — 一覧はラベルだけ、マウスを乗せると全文が出る
- 各行に編集（鉛筆）・削除（ゴミ箱）アイコン
- クリップボードから登録 — いまコピー中の文字列をワンボタンで定型文化
- 鍵（機密モード） — 住所などは鍵アイコン付き。クリックで開錠 → もう一度でコピー（誤コピー防止）
- プレースホルダ展開 — `{{date}}`/`{{time}}` は自動、`{{宛名}}` 等はコピー時に入力
- テーマカラー選択 — 設定から数色のプリセットを切り替え
- グローバルホットキー — 好きなキーでウィンドウを最前面に呼び出し
- トレイ常駐 — ×でトレイにしまい、アイコンのクリック／右クリック（表示・終了）で操作
- 完全オフライン — データは手元の templates.json にのみ保存。外部送信なし

機密情報を含む定型文も、自分の PC 内で完結して扱えます。

## 使い方（Python で実行）

```bash
python stamper.py
```

Python 3.10 以降を推奨。実行に追加ライブラリは不要です（Tkinter は標準同梱）。
※ アイコン再生成とビルドにだけ Pillow / PyInstaller を使います。

## テスト / 品質ゲート

純粋ロジック（色のコントラスト・ホットキー解析・プレースホルダ展開・データの保存/破損時フォールバック・ラベル整形など）を `tests/` のユニットテストで検証します。GUI は起動しないのでディスプレイ無しでも通ります。

```bash
python -m unittest discover -s tests -p "test_*.py" -v
```

- **ビルド時**：`build.bat` は最初にテストを実行し、**全部通らなければ exe をビルドしません**。
- **CI**：`main` への push / PR で GitHub Actions がテストを実行（`.github/workflows/ci.yml`）。
- **リリース**：タグ `v*` を push すると、テスト合格時のみ exe をビルドしてリリースへ添付（`.github/workflows/release.yml`）。テストが落ちたらリリース成果物は作られません。

## exe をビルドする

`build.bat` をダブルクリックするだけ。テストが通れば `dist\Grimoire.exe` が生成されます。

手動で実行する場合:

```bash
pip install pyinstaller pillow
python make_icon.py            # icon.ico を生成（同梱済みなら不要）
pyinstaller --onefile --windowed --name Grimoire ^
            --icon icon.ico --add-data "icon.ico;." stamper.py
```

> exe と同じフォルダに `templates.json` が作られます（ポータブル運用）。
> 別 PC へ移すときは、必要に応じて `templates.json` も一緒にコピーしてください。

## 操作一覧

| 操作 | 方法 |
|------|------|
| コピー（通常） | 行をクリック／Enter／番号キー 1〜9 |
| コピー（鍵付き） | 鍵行をクリック→開錠→もう一度クリック（誤コピー防止） |
| 行を選ぶ | ↑↓キー（Enter でコピー） |
| お気に入り登録／解除 | 行の★をクリック（「お気に入り」タブに集約） |
| 全文を見る | 行にマウスを乗せる（ホバー） |
| 定型文の並び替え | 行の左のグリップ（⋮⋮）をドラッグ |
| 編集 | 行の鉛筆アイコン |
| 削除 | 行のゴミ箱アイコン |
| 追加 | ツールバー「＋ 定型文を追加」 |
| クリップボードから追加 | ツールバー「クリップボードから登録」 |
| 検索（全フォルダ横断） | 右上の検索ボックスに入力（✕ で消去・Esc でも消去） |
| フォルダ追加 | タブ右の「＋」／タブ空き位置をダブルクリック／メニュー「フォルダ」 |
| フォルダ名変更 | タブをダブルクリック（または右クリック） |
| フォルダ並び替え | タブをドラッグ |
| フォルダ削除 | タブを右クリック→削除（最大10個まで作成可） |
| トレイにしまう | ウィンドウの×（アイコンのクリック／ホットキーで復帰） |
| 終了 | トレイアイコンを右クリック →「終了」 |
| 設定（色・ホットキー等） | メニュー「設定」 |

### 設定

メニュー「設定」から変更できます。

- テーマカラー：数色のプリセットから選択
- ホットキー：`修飾キー + キー`（例: `ctrl+alt+space`, `ctrl+f1`）。修飾キーは `ctrl/alt/shift/win`。
  ※ Catmemonote の `Ctrl+CapsLock` や clibor の `Ctrl×2` と被らないものを選んでください（Windows 専用）。
- 直接貼り付け：コピー後に、呼び出す前のアプリへ自動で Ctrl+V（オン/オフ）
- 削除の前に確認する：オン/オフ
- Windows起動時に自動で起動（タスクトレイに常駐）：オン/オフ
  - レジストリを汚さず、`%APPDATA%\Microsoft\Windows\Start Menu\Programs\Startup\Grimoire.lnk` を置く/消すだけ。配布版（exe）からの起動時のみ設定可。

### プレースホルダ

本文に書くと、コピー時に展開されます。

- `{{date}}` → 今日の日付（例 2026-05-21）
- `{{time}}` → 現在時刻（例 14:30）
- `{{datetime}}` → 日付＋時刻
- `{{宛名}}` など上記以外 → コピー時に入力欄が出る

## データ形式

`templates.json`:

```json
{
  "settings": { "hotkey": "ctrl+alt+space", "accent": "#7E57C2", "confirm_delete": true },
  "folders": [
    {
      "name": "メール定型",
      "items": [
        { "title": "書き出し", "body": "お世話になっております。", "secret": false }
      ]
    }
  ]
}
```

手で編集してもOKです（`title` は省略可。空なら本文の先頭が一覧に表示されます）。
`"secret": true` の項目が鍵付き（クリックで開錠 → もう一度でコピー）になります。
